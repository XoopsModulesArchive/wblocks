Installation:
-------------
1. Decompress and put this folder somewhere inside your document root
2. Upload the wblocks directory to XOOPS_ROOT_URL/modules/
3. Log in to your site as administrator.
4. Install the module using the XOOPS module administration panel at 'System Admin -> Modules'.
5. Done!
